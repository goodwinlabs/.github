# Welcome to Goodwin Labs 👋

This is the home of my personal GitHub organization, where I bring together all things related to infrastructure as code, automation, and personal productivity. Here, you'll find repositories that power my homelab, manage my dotfiles, and provide utilities for streamlining workflows.

## What's Going to be Here?

- **Infrastructure as Code**: Repositories showcasing my homelab setup, built with tools like Terraform, Ansible, and Kubernetes.
- **Dotfiles**: My personal configuration files for tools like Vim, Zsh, and Tmux, designed to make development efficient and enjoyable.
- **Utilities**: Handy scripts and tools for automating tasks, managing systems, and experimenting with new ideas.

## Why Follow?

If you're interested in:
- Learning how to manage a homelab with code,
- Exploring well-documented dotfiles for inspiration,
- Discovering utilities that make life easier for developers and sysadmins,

...then you're in the right place! Feel free to explore, fork, and contribute.
